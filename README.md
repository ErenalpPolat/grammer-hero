# 🦉 Grammar Hero

İngilizce dilbilgisini Duolingo tarzı oyunlaştırılmış bir deneyime dönüştüren web
uygulaması. Günde 5 dakikada, oyun oynar gibi gramer öğren.

**Live demo:** _deploy edildiğinde burası dolar_

---

## Özellikler

- 🎯 **17 ünite · 102 ders · ~4.000 soru** — Present Simple'dan Phrasal Verbs'a
  kadar İngilizce gramerin tamamı
- 🎮 **8 farklı oyun tipi** — aynı konuyu farklı şekillerde öğren, sıkılma
- 🔥 **Streak sistemi** — her gün çalış, ateşini söndürme
- ⭐ **XP · seviye · kalp · gem** — Duolingo-vari gamification
- 🏆 **Lig sistemi** — 10 tier (Bronz → Elmas) · haftalık 15 kişilik gruplarda
  yarış, yüksel
- 🧠 **Akıllı tekrar** — yanlış yaptığın sorular otomatik tekrar defterine
  düşer, SM-2 algoritmasıyla seyrekleşerek hatırlatılır
- 🎯 **Günlük hedef** — 5/10/15/20 dakikalık hedef, progress takibi
- 🌙 **Karanlık mod** · tam Türkçe arayüz · ücretsiz · reklamsız

---

## 8 Oyun Tipi

| Tip | Mekanik |
|---|---|
| **Çoktan Seçmeli** | 4 şıktan doğru olanı seç |
| **Fill in the Blank** | Cümle boşluğuna doğru kelimeyi yerleştir |
| **True / False** | Cümle gramer olarak doğru mu? |
| **Cümle Kur** | Karışık parçaları doğru sıraya diz |
| **Kelime Bul** | Karışık harflerden kelime oluştur |
| **Hatayı Bul** | Cümledeki yanlış kelimeyi tıkla |
| **Sürükle-Bırak** | Kelimeyi boşluğa sürükle |
| **Eşleştir** | Memory match — eşleştir ve bul |
